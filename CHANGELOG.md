# Changelog

All notable user-facing changes to this project should be recorded here.

## v0.2.0

- Split the main screen into dedicated `Knowledge` and `Flashcards` tabs so tree editing and review are no longer cramped into one view.
- Added a focused flashcards dashboard with due-card actions, scheduled-card lists, and clearer review entry points.

## v0.1.0

- Bootstrapped the Knowledge app as a tree-based note board with nested nodes, drag-to-nest organization, and a markdown-style detail view.
- Added flashcard tagging with due-card review actions (`Forgot`, `Hard`, `Good`, `Easy`) and stored review scheduling.
- Added local JSON persistence, automatic rolling backups, clipboard export, pasted-JSON import, and changelog/version wiring.
